# AI Multi-Agent Resume Intelligence System

## Overview

Built a Multi-Agent Resume Analyzer using Local LLM and RAG architecture.

## Features

* Resume Analyzer Agent
* Job Match Agent
* Resume Improvement Agent
* Local AI (Ollama + Llama3)
* Vector Database (FAISS)
* RAG Pipeline

## Tech Stack

* Flowise
* Ollama
* Llama3
* FAISS
* Node.js

## Architecture

PDF → Embedding → Vector Store → Multi-Agent → Chat Interface

## Use Cases

* Resume Analysis
* Job Matching
* Resume Optimization
